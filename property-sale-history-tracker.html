📖 Tool kaise use karein (Hindi Guide)

  1. Property Add karein: "+ Add Property" button par click karke property ka address, type (Residential/Commercial/Plot), size aur original purchase details bharein.
  2. Sale History Add karein: Har property card open karke "+ Add Sale Record" se naya sale entry add karein — seller, buyer, sale price aur date ke saath. Tool automatically profit/loss calculate kar dega previous record se compare karke.
  3. Resale Chain Track karein: Ek property kai baar bik sakti hai. Har baar naya record add karne se timeline mein pura resale history dikhega — kisne kab kitne mein li aur kitne mein becha.
  4. Search & Filter: Upar diye search box se kisi bhi address, owner ya buyer ka naam search karein. Property type aur price ke hisaab se sort bhi kar sakte hain.
  5. Dashboard Stats: Top par diye gaye cards mein total properties, total sale value, total profit aur average appreciation% dikhta hai — pure business ka overview ek nazar mein.
  6. Export/Print: "Export CSV" button se sara data Excel mein download karein, ya "Print" se report nikal sakte hain client ko dikhane ke liye.
  7. Data Safe Hai: Sara data aapke browser ke LocalStorage mein save hota hai — refresh karne par bhi data delete nahi hoga. (Note: browser cache clear karne se data delete ho sakta hai, isliye regular CSV export karte rahein.)
  8. Property Delete karein: Kisi property ya sale record ko delete karne ke liye uske bagal mein diye red "Delete" button ka use karein.
